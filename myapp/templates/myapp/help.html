Getting Started

1. Manage CardsUse Quick Tools on Home to add/import/export cards. Turn on Edit Cards beside Search to edit fields, press Save to apply, or turn it off to cancel unsaved changes.
2. Open DetailsClick any card title to open full post details and related posts.
3. Find FasterUse search + tag filters + sorting + date range filters on Home and Collections.
4. Chat and ShareOpen Chat from sidebar or friend list and share cards directly into conversations.
5. Message ControlsDelete for me hides only on your screen, while Delete for all removes for both users.
6. Message StatusSingle tick = sent, double tick = delivered, blue double tick = read. Red dots show unread messages.
7. Admin EditingAdmin Center has separate edit toggles for User Management and Post Moderation tables. Save changes with each section Save button, or turn edit mode off to cancel unsaved edits.
8. Navigate PagesUse sidebar items like Dashboard, Friends, Chat, AI Helper, Reports, and Favourites.
9. Dark ModeSet theme in Settings, or use the quick Dark Mode button in the right panel below the Friends box.
10. Theme PersistenceTheme persists within the current open browser session (including logout/login). After closing all browser windows and reopening, mode resets to normal light.
11. SecuritySessions are tab-scoped for safety, so reopening the app in a new browser session can require login again.
12. Login MemoryUse Remember me on login to keep account session for longer; if unchecked, session closes with browser. This device remembers previously used usernames and browser autofill shows native account/password suggestions.
13. AI Helper ButtonUse the large round AI button on the right-bottom side to open assistant chat. It stays open until you press Close.
14. CustomizeSet your default sort in Settings and update display name in Profile.
